Compute Engine Released
2026-03-04

📋 概要

Google Compute Engine のゲストエージェント v20260228.00 がリリースされた。 2026 年 3 月 3 日に Linux 6 ディストリビューション向けに配信が開始され、3 月 4 日に Windows 向けにも配信が開始された。

ゲストエージェントは Compute Engine のゲスト環境の中核コンポーネントであり、VM インスタンス上で ネットワーク設定、SSH アクセス管理、メタデータアクセス、OS Login 統合などの重要な機能を担っている。 v20250901.00 以降はプラグインベースのアーキテクチャに移行しており、 コアプラグインとオプションの拡張機能 (エクステンション) に分離された設計となっている。

✔ プラグインの分離実行により、1 つのプラグインのクラッシュが他に影響しない
✔ OS レベルのリソース制限により、CPU / メモリの過剰消費を防止

💡 ユースケース

🛡️ VM セキュリティ管理

OS Login 統合により、IAM ロールベースの SSH アクセス管理を実現。個別の SSH 鍵管理が不要になり、セキュリティポリシーの一元管理が可能。

💻 マルチ OS 環境の運用

Linux と Windows の両方をサポートし、統一されたゲストエージェント管理が可能。混在環境での運用負荷を軽減。

🔍 オブザーバビリティ統合

Ops Agent エクステンションを通じて Cloud Monitoring / Cloud Logging と連携し、VM のメトリクスとログを自動収集。

📝 まとめ・推奨アクション

ゲストエージェント v20260228.00 は、Linux 6 ディストリビューションおよび Windows を対象とした定期リリースである。

💡 推奨: Compute Engine VM を運用している場合は、自動更新が有効でない環境では手動でのアップデートを検討されたい。
✅ 自動更新の有効化: google-compute-engine-auto-updater パッケージをインストールすることで、ゲスト環境パッケージの自動更新を有効化できる。
Bash 
# 自動更新パッケージのインストール (Debian/Ubuntu)
sudo apt-get install google-compute-engine-auto-updater

# ゲストエージェントのバージョン確認
google_guest_agent --version
