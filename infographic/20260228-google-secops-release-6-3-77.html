SIEM & SOAR - 全リージョンで利用可能に

📋 概要

Google SecOps の Release 6.3.77 が全リージョンで利用可能になりました。2026 年 2 月 22 日から段階的に展開が進められ、2 月 28 日に全リージョンでの提供が完了しています。

本リリースの最も重要な変更点は、Publisher Agent Version 2.6.4 において、リモートエージェントでの Python 3.7 サポートが削除されたことです。リモートエージェントを利用しているユーザーは Python 3.11 への移行が必要です。

✕ Python 3.7 は非推奨だが動作していた

✓ Python 3.11 が推奨バージョン

⚠️ デメリット・制約事項

⚠️ 注意: 以下の制約事項を把握した上で、移行計画を立ててください。

✕ Python 2.7 および 3.7 向けに構築されたインテグレーションでは、Integration Rollback 機能がサポートされない

🎯 ユースケース

🏢 オンプレミス環境のセキュリティ監視

リモートエージェントを通じてオンプレミスのログやイベントを Google SecOps に送信し、クラウドベースの脅威検出エンジンで分析を実施。Python 3.11 への移行で最新のセキュリティパッチが適用される。

🔄 自動インシデントレスポンス

SOAR プレイブックと連携し、リモート環境でのインシデント対応を自動化。Publisher Agent が Python 3.11 で動作することで、最新のインテグレーションとの互換性が確保される。
