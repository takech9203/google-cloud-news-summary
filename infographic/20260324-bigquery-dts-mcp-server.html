📄 Overview

BigQuery Data Transfer Service に、リモート Model Context Protocol (MCP) サーバーが新たに追加された。 AI エージェントがデータ転送の作成、管理、実行を自然言語で操作できるようになる。

🤖 MCP とは: Anthropic が開発したオープンソースプロトコル。LLM / AI アプリケーションが外部データソースに標準化された方法で接続するための仕組みを提供する。Google Cloud は BigQuery、GKE、Cloud SQL、Spanner など多数のサービスで MCP サーバーを既に提供しており、BigQuery Data Transfer Service もこのエコシステムに参加。

🔄 Before / After

❌ Before

  • Google Cloud Console / bq CLI / REST API を直接使用する必要があった
  • AI エージェント連携には独自インテグレーションの構築が必要
  • 転送設定やステータス確認に複数ツール/インターフェースを切り替え

✅ After

  • AI エージェントが MCP 経由でデータ転送を直接操作可能
  • 自然言語による指示で転送の設定・管理が可能
  • IAM / OAuth 2.0 / Model Armor / 監査ログなどエンタープライズ機能を標準搭載

🛠️ Architecture

AI アプリケーション内の MCP クライアントが、リモート MCP サーバーを介して BigQuery Data Transfer Service を操作し、各種データソースから BigQuery へのデータ転送を制御する。

⚡ Key Features

🚀 1. MCP Protocol Data Transfer Management
  • Transfer Configuration (転送設定) の作成 / 更新 / 削除を MCP ツール経由で実行
  • Transfer Run (転送実行) の開始、ステータス確認、履歴取得
  • AI エージェントが自然言語の指示を MCP ツール呼び出しに変換
🔒 2. Enterprise Security
  • OAuth 2.0 + IAM による認証・認可
  • MCP Discovery: tools/list メソッドでツール自動検出
  • Model Armor による入出力セキュリティスキャン (オプション)
  • 監査ログの集中管理
🔄 3. Backward Compatibility
  • Google Cloud Console / bq CLI / REST API は引き続き利用可能
  • MCP サーバーは新たなアクセスチャネルとして追加

🛠️ Setup

Prerequisites

  1. BigQuery Data Transfer Service API が有効化済み
  2. MCP サーバーがプロジェクトで有効化済み
  3. 適切な IAM ロール (roles/bigquery.admin or necessary permissions) が付与済み

Step 1: Enable MCP Server

Bash MCP Server Activation 
gcloud beta services mcp enable bigquerydatatransfer.googleapis.com \
  --project=PROJECT_ID

Step 2: MCP Client Configuration

JSON MCP Client Settings (Gemini CLI / Claude Code) 
{
  "mcpServers": {
    "bigquery-data-transfer": {
      "url": "https://bigquerydatatransfer.googleapis.com/mcp",
      "transport": "http"
    }
  }
}

Authentication details depend on the AI application being used. See MCP Authentication docs for more information.

⚖️ Merits & Considerations

✅ Business Merits

  • Operational Efficiency: AI agent automation improves data engineer productivity
  • Natural Language Operations: No complex API / CLI knowledge required

✅ Technical Merits

  • Standard Protocol: MCP open standard enables broad AI app compatibility
  • Enterprise Security: IAM, OAuth 2.0, Model Armor, audit logs via existing Google Cloud infrastructure

⚠️ Limitations

  • Preview status: SLA is not applicable
  • Pre-GA Offerings Terms apply; limited support

⚠️ Considerations

  • AI agent operations may cause unintended config changes; strict IAM controls are essential
  • Not recommended for production environments during Preview

🎯 Use Cases

💻 UC1: AI Agent Data Transfer Setup

Scenario: A data engineer needs to set up a new data transfer from a SaaS data source to BigQuery. Using Gemini CLI or Claude Code agent mode, they create a transfer configuration with natural language instructions.

⚡ Effect: Transfer setup time is reduced, eliminating the need to reference API documentation.

🔬 UC2: Transfer Monitoring & Incident Response

Scenario: An AI agent periodically checks transfer run status and triggers a backfill when a failed transfer is detected.

⚡ Effect: Automated detection and recovery of transfer failures helps maintain data freshness.

💰 Pricing

🔸 Note: MCP server itself has no additional charges documented at this time. Standard BigQuery Data Transfer Service pricing applies. After data is transferred, standard BigQuery storage and query costs apply.
