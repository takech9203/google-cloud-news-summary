📋 概要

Google Cloud CCaaS バージョン 4.16 がリリースされた。仮想エージェントの SIP リングタイムアウト設定、Agent Assist のキュー非関連対応、HubSpot Help desk ビュー、事前録音オーディオ再生 の 4 つの新機能と、12 件のバグ修正を含む包括的なアップデートである。

コンタクトセンター運用者、CRM 管理者、仮想エージェント開発者にとって、運用安定性と機能拡張の両面で重要なアップデートとなる。

🔄 Before / After 比較

❌ アップデート前の課題

  • SIP 転送時のリングタイムアウト設定不可。応答前に切断されるケースあり
  • Agent Assist はキュー関連インタラクションのみ利用可能
  • HubSpot は Standard ビューのみ
  • 仮想エージェントは TTS のみで事前録音オーディオ不可
  • 12 件のバグにより録音欠落、メール不具合、チャット転送失敗が発生

✅ アップデート後の改善

  • sip_ring_timeout で最大 600 秒のタイムアウト設定が可能に
  • Agent Assist がチームレベルで有効化可能。直接インバウンド・アウトバウンドにも対応
  • HubSpot で Standard / Help desk ビューの切り替えが可能に
  • Dialogflow が事前録音オーディオで応答可能。全音声チャネル対応
  • 12 件のバグ修正でプラットフォーム全体の安定性が向上

✨ 4 つの新機能

📞

SIP リングタイムアウト設定

Twilio ユーザー向け。仮想エージェントのカスタムペイロードに sip_ring_timeout を追加し、SIP エンドポイントへの転送時のリング期間を最大 600 秒まで設定可能。内部内線や UC 宛先への転送成功率が向上。

🧠

Agent Assist キュー非関連対応

Agent Assist をチームレベルで有効化可能に。キューに関連付けられていないインタラクション (直接インバウンド、アウトバウンド) でも利用可能。Settings > Users & Teams から設定。

🎫

HubSpot Help desk ビュー

HubSpot 連携で CRM チケットビューを選択可能に。従来の Standard ビューに加え、リアルタイムの Help desk ビューが利用可能。Settings > Developer Settings > HubSpot から設定。

🔊

事前録音オーディオ再生

Dialogflow が仮想エージェントの応答として事前録音オーディオを再生可能に。標準 TTS に代わり高品質オーディオを使用。全音声チャネル (インバウンド / アウトバウンド) に対応。

🛠️ 設定方法

💡 前提条件: Google Cloud CCaaS インスタンスが稼働中であり、CCaaS 4.16 へのアップデートが適用されていること (デプロイメントスケジュールに依存)。
1

SIP リングタイムアウトの設定 (Twilio ユーザー)

Dialogflow CX コンソールで仮想エージェントのカスタムペイロードに sip_ring_timeout フィールドを追加。値は秒単位で最大 600 を指定。

2

Agent Assist のチームレベル有効化

CCAI Platform ポータルで Settings > Users & Teams > Manage Users & Teams から対象チームを編集し、Agent Assist セクションで機能を有効化。

3

HubSpot CRM チケットビューの変更

Settings > Developer Settings > HubSpot パネルの CRM Ticket View セクションで Standard / Help desk ビューを選択。

4

事前録音オーディオの設定

Dialogflow CX のフルフィルメントオプションで事前録音オーディオを設定。オーディオファイルはモノラルの mu-law エンコード (8kHz) で Cloud Storage にホスト。

📈 メリット

💼 ビジネス面

  • 顧客体験の向上: 事前録音オーディオでブランドに合った高品質な音声応答を提供
  • エージェント支援の拡大: Agent Assist がキュー非関連のインタラクションでもリアルタイム支援可能に
  • CRM 運用効率の改善: HubSpot Help desk ビューでチケット管理のリアルタイム性が向上

🔧 技術面

  • SIP 転送の信頼性向上: リングタイムアウトのカスタマイズで転送成功率が向上
  • プラットフォーム安定性: 12 件のバグ修正で録音、メール、チャット転送、API の安定性が改善
  • 柔軟なオーディオ対応: TTS に加え事前録音ファイルで音声品質を制御可能

⚠️ 制約事項・注意点

⚠️ 制限事項:
  • SIP リングタイムアウトは Twilio ユーザーのみが利用可能 (BYOC 等の他テレフォニー統合では未対応)
  • sip_ring_timeout の最大値は 600 秒に制限
  • 事前録音オーディオはモノラルの mu-law エンコード (8kHz) が必要。Cloud Storage にホストが必要
💡 考慮すべき点:
  • CCaaS 4.16 の適用タイミングはデプロイメントスケジュールに依存し、即時反映ではない場合がある
  • Agent Assist のチームレベル設定は既存のキューレベル設定と併用するため、設定の整合性確認が必要
  • HubSpot Help desk ビューは新機能のため、既存ワークフローとの互換性を事前確認推奨

💡 ユースケース

📞 UC 連携のリングタイムアウト最適化

仮想エージェントが IVR 処理後に社内 UC システムの内線に転送するケースで、UC 側の応答に時間がかかり転送が切断されていた。

JSON 
{
  "ujet": {
    "type": "action",
    "action": "deflection",
    "deflection_type": "sip",
    "sip_uri": "sip:internal-ext-1234@uc.example.com",
    "sip_ring_timeout": 180
  }
}

✅ リングタイムアウトを 180 秒に設定し、UC 側エージェントの応答猶予を確保。転送成功率が向上。

📊 アウトバウンドセールスでの Agent Assist 活用

セールスチームがキューを選択せずにアウトバウンドコールを行う際、これまで Agent Assist が利用できなかった。

✅ チームレベルで Agent Assist を有効化し、アウトバウンドセールスコール中にリアルタイムの提案やナレッジベース情報を表示。商談の質が向上。

🎤 ブランド統一の音声体験

金融機関のコンタクトセンターで、仮想エージェントの TTS 音声がブランドイメージと一致しない。

✅ プロのナレーターによる事前録音オーディオを使用し、ブランドに合った一貫性のある音声体験を提供。サポート、タスク、ポストセッションの各仮想エージェントで利用可能。
