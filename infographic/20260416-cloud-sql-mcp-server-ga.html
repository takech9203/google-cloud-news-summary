☁️ Cloud SQL: リモート MCP サーバーが GA

Cloud SQL MySQL PostgreSQL SQL Server GA
2026-04-16

📋 概要

Cloud SQL のリモート Model Context Protocol (MCP) サーバーが一般提供 (GA) となりました。LLM や AI アプリケーションから Cloud SQL インスタンスと容易にやり取りできるマネージドサービスです。MySQL、PostgreSQL、SQL Server の全 3 エンジンに同時適用されます。

MCP は Anthropic が開発したオープンソースプロトコルで、AI アプリケーションが外部データソースに接続するための標準化された仕組みを提供します。Google のインフラストラクチャ上で動作し、HTTP エンドポイント (https://sqladmin.googleapis.com/mcp) を通じて AI アプリケーションとの通信を行います。

✅ GA リリースのポイント: 本番環境での使用が正式にサポートされ、SLA に基づいたサービス品質保証が提供されます。自然言語でデータベースの作成、クエリ実行、スキーマ変更などを行える環境が本番グレードで利用可能になりました。

🔄 Before / After 比較

❌ Before

  • Cloud SQL の管理には Console、gcloud CLI、REST API の直接操作が必要で AI からの統合操作が困難
  • LLM / AI エージェントからの操作には独自の API ラッパーや中間レイヤーの構築が必要
  • MCP サーバーはプレビュー段階で SLA なし、サポートも限定的

✅ After

  • Gemini CLI、Claude、ChatGPT などから自然言語で Cloud SQL を直接操作可能
  • GA リリースにより SLA 適用、本番ワークロード利用を正式サポート
  • OAuth 2.0 / IAM / Model Armor による多層セキュリティ保護が利用可能

🔧 設定方法

前提条件

→Google Cloud プロジェクトが作成済みであること
→対象の Cloud SQL API (MySQL / PostgreSQL / SQL Server) が有効化されていること
→適切な IAM ロールが付与されていること

Gemini CLI での設定

Bash エクステンションディレクトリの作成 
mkdir -p ~/.gemini/extensions/cloud-sql/

以下の内容で ~/.gemini/extensions/cloud-sql/gemini-extension.json を作成:

JSON gemini-extension.json 
{
  "name": "cloud-sql",
  "version": "1.0.0",
  "mcpServers": {
    "Cloud SQL MCP Server": {
      "httpUrl": "https://sqladmin.googleapis.com/mcp",
      "authProviderType": "google_credentials",
      "oauth": {
        "scopes": ["https://www.googleapis.com/auth/cloud-platform"]
      },
      "timeout": 30000,
      "headers": {
        "x-goog-user-project": "PROJECT_ID"
      }
    }
  }
}

Claude での設定 (Enterprise / Team プラン)

1.Google Cloud Console で OAuth 2.0 クライアント ID を作成
2.承認済みリダイレクト URI に https://claude.ai/api/mcp/auth_callback を追加
3.Claude.ai の Admin settings > Connectors でカスタムコネクタを追加
4.サーバー URL に https://sqladmin.googleapis.com/mcp を設定

HTTP リクエストでの直接利用

Bash ツール一覧の取得 (認証不要) 
curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc": "2.0", "id": 0, "method": "tools/list"}' \
  "https://sqladmin.googleapis.com/mcp"
Bash ツール呼び出し (認証必要) 
curl -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $(gcloud auth print-access-token)" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "tools/call",
    "params": {
      "name": "list_instances",
      "arguments": {
        "project": "PROJECT_ID"
      }
    }
  }' \
  "https://sqladmin.googleapis.com/mcp"

✅ メリット

💼 ビジネス面

→開発生産性の大幅向上: 自然言語で DB 作成、クエリ実行、スキーマ変更が可能。SQL 非習熟者も参加可能
→運用コスト削減: AI エージェントによるルーティン作業の自動化
→本番環境での信頼性: SLA 適用、エンタープライズ要件を満たす品質保証

🔬 技術面

→標準プロトコル: MCP はオープンスタンダード、ベンダーロックインなし
→マネージドインフラ: Google がホスト、ローカルサーバー構築不要
→多層セキュリティ: OAuth 2.0 / IAM / Model Armor

⚠️ デメリット・制約事項

制限事項

  • ⚠️ SQL Server では execute_sql、create_user、update_user ツールが利用不可
  • ⚠️ execute_sql のレスポンスが 10 MB を超える場合、結果が切り捨て
  • ⚠️ execute_sql で 30 秒を超えるクエリはタイムアウトの可能性
  • ⚠️ create_user ではビルトイン認証 (パスワード) のユーザー作成は非対応
  • ⚠️ execute_sql の利用には IAM データベース認証の有効化と data_api_access = ALLOW_DATA_API の設定が必要

考慮すべき点

  • 💡 API キー認証は非対応、OAuth 2.0 が必須
  • 💡 AI エージェントには専用サービスアカウント + 最小権限の原則を推奨
  • 💡 Model Armor は自然言語データを含む MCP トラフィックがある場合にのみ有効化を推奨
  • 💡 リモート MCP サーバーとローカル MCP サーバー (MCP Toolbox for Databases) はユースケースに応じた選択が必要

🎯 ユースケース

💻 ユースケース 1: AI アシスタントによる Web アプリ開発

開発者が新しい Web アプリケーションのバックエンド DB を自然言語で素早くセットアップ。

プロンプト: 「MySQL の開発用インスタンスを作成して、products テーブルをセットアップしてください」
  1. create_instance でインスタンスを作成
  2. get_operation で作成完了を確認
  3. get_instance で接続情報を取得
  4. execute_sql で CREATE TABLE を実行
  5. execute_sql で初期データを INSERT

効果: プロビジョニングからスキーマ設計、初期データ投入までを会話形式で完了。開発イテレーションが大幅に加速。

📊 ユースケース 2: 自然言語によるデータベースクエリと分析

ビジネスアナリストが SQL を書かずに、自然言語でデータを検索・分析。

プロンプト: 「在庫テーブルから価格が100ドル以上の靴の一覧を表示してください」
  1. execute_sql で適切な SELECT 文を生成・実行
  2. 結果を人間が読みやすい形式で返却

効果: SQL 知識不要でデータ取得が可能に。データドリブンな意思決定を加速。

⚙️ ユースケース 3: AI エージェントによる DB 運用自動化

運用チームが AI エージェントを使って Cloud SQL の日常管理タスクを自動化。

プロンプト: 「全インスタンスの一覧を表示し、開発環境のインスタンスを最新設定に更新してください」
  1. list_instances で全インスタンスを取得
  2. get_instance で各インスタンスの詳細を確認
  3. update_instance で該当インスタンスの設定を更新
  4. get_operation で更新完了を確認

効果: 定型的な運用タスクを自然言語で指示するだけで実行。運用効率が向上。

💰 料金

項目料金
MCP サーバーへのアクセス無料 (Cloud SQL API の一部)
Cloud SQL インスタンス (MySQL)vCPU とメモリに応じた従量課金
Cloud SQL インスタンス (PostgreSQL)vCPU とメモリに応じた従量課金
Cloud SQL インスタンス (SQL Server)vCPU、メモリ、ライセンスに応じた従量課金
Model Armor (オプション)リクエスト数に応じた従量課金

🌍 利用可能リージョン

Cloud SQL リモート MCP サーバーはグローバルエンドポイント (https://sqladmin.googleapis.com/mcp) として提供されており、Cloud SQL インスタンスが利用可能な全リージョンで使用できます。

🔗 関連サービス・機能

☁️ Google Cloud MCP エコシステム

BigQuery、Cloud Storage など他の Google Cloud サービスもリモート MCP サーバーを提供。統合的な AI 駆動のクラウド管理が可能。

🧰 MCP Toolbox for Databases

Cloud SQL 向けのローカル MCP サーバーソリューション。ローカル環境での実行が必要な場合に利用。

🛡️ Model Armor

MCP ツール呼び出しとレスポンスのセキュリティスキャン。プロンプトインジェクションや機密データ漏洩を防止。

📝 Cloud SQL Studio

ブラウザベースの SQL エディタ。MCP 経由ではなく直接 SQL を実行する場合に利用。
