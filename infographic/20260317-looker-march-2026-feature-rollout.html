🔍 概要

Looker の大規模な機能ロールアウトが開始。Gemini AI の全面的な統合とセルフサービス分析機能の成熟を象徴するアップデートで、データアナリスト、BI 開発者、Looker 管理者の全レイヤーに影響する。

🤖 Gemini AI が Looker に全面統合: Visualization Assistant、Quick Start Analyses、Expression Assistant、Enhanced Search、LookML Parameter Assistant、Conversational Analytics の 6 機能で Gemini を活用

Before / After

❌ Before

  • ビジュアライゼーション変更に JSON 手動編集が必要
  • テーブル計算に Looker Expression の専門知識が必要
  • Self-service Explores は Preview 段階
  • コンテンツ認証機能が限定的
  • Explore の行数制限が 5,000 行
  • 検索はキーワードベースの単純マッチング

✅ After

  • 自然言語でビジュアライゼーションをカスタマイズ (GA)
  • Gemini が Looker Expression の記述を支援 (Preview)
  • Self-service Explores が GA + Google Sheets 対応
  • Content Certification が GA + 自動認証機能
  • 行数制限を最大 50,000 行まで設定可能 (Preview)
  • Gemini によるセマンティック検索 (Preview)

✅ GA 機能 (3 機能)

GA 🤖 Visualization Assistant

Gemini を活用し、自然言語プロンプトで Looker ビジュアライゼーションのフォーマットオプションをカスタマイズ。テキストベースのプロンプトから JSON フォーマットオプションを生成し、ビジュアライゼーションに適用する。テンプレートやパターンの作成起点としても利用可能。

必要な権限: can_override_vis_config 権限を持つ Looker ロール + Looker Assistants 設定の有効化
GA 📊 Self-service Explores

LookML モデルの設定や Git バージョン管理なしで、データファイルをアップロードして Explore でクエリ・可視化が可能。CSV、Excel (XLS/XLSX)、Google Sheets に対応。新たに Google Drive ナビゲーション (OAuth) を使った Google Sheets からのデータアップロードをサポート。アップロードデータは BigQuery に永続化される。

必要な権限: upload_data 権限 / Google Sheets アップロードには Looker 26.2 以降が必要
GA ✅ Content Certification

ダッシュボード、Look、Self-service Explores に加えて、LookML Explores の認証が可能に。管理者が現在および将来のすべての LookML ダッシュボードと Explores を自動認証する機能を追加。認証ステータスに基づくコンテンツのソート機能が Enhanced Search に統合。

→認証済みコンテンツには緑色の「Certified」バッジが表示 →認証されていない Self-service Explores には「ungoverned」バッジが表示 →certify_content 権限を持つ Looker 管理者が認証操作を実行

🔬 Preview 機能 (5 機能)

Preview ⚡ Quick Start Analyses

Gemini が Explore に対して Quick Start 分析を自動生成。ユーザーがデータ探索の出発点として利用でき、セルフサービス分析の導入障壁を下げる。

Preview 📝 Expression Assistant

Gemini を活用して Looker Expression (テーブル計算やカスタムフィールド) の記述を支援。自然言語で計算ロジックを説明するだけで、適切な Looker Expression を提案。

Preview 🔍 Enhanced Search

Gemini を活用して検索クエリの概念的な意味を解釈し、キーワードマッチングを超えたセマンティック検索を実現。Content Certification と統合され、認証ステータスによるコンテンツの絞り込みが可能。

Preview 📅 Custom Calendars

Snowflake または BigQuery 接続において、会計カレンダーやリテールカレンダーをサポート。企業独自の会計年度やリテール業界の特殊なカレンダー体系に対応。

Preview 📈 Increased Row Limit

管理者が地図チャート、散布図チャート、テーブルチャートの行数制限を最大 50,000 行まで設定可能。従来の 5,000 行制限から大幅に緩和。

🔧 その他の改善 (4 機能)

🔌 Disable Connection オプション

Connections Settings ページに新しいオプションが追加。ダウンストリームのデータベース障害発生時に接続を無効化し、障害切り分けやメンテナンス時に接続を一時的に停止可能。

💡 LookML Parameter Assistant

自然言語プロンプトに基づいて LookML パラメータを生成する機能。個別に有効化できるようになり、管理者が必要な機能のみ選択的に導入可能。

🔄 CI Feature Update

Continuous Integration が新バージョンに移行。UI 統合が改善され、より直感的な CI ワークフローを提供。

💬 Conversational Analytics モード追加

Fast モード: より迅速な回答を提供し、シンプルな質問に最適。Thinking モード: 複雑な質問に対して深い分析と推論を行い、より正確な回答を生成。

⚠️ デメリット・制約事項

制限事項:
  • Preview 機能は「Pre-GA Offerings Terms」に基づき提供され、サポートが限定的
  • Gemini in Looker の Preview 機能は現在無料だが、期間終了後は追加料金の可能性あり
  • Google Sheets アップロードは Looker 26.2 以降が必要
  • Self-service Explores のデータは BigQuery に書き込まれ、BigQuery 管理者が閲覧可能
  • Increased Row Limit は地図、散布図、テーブルチャートのみが対象
考慮すべき点:
  • Conversational Analytics は FedRAMP High / Medium の認可境界に含まれていない
  • Self-service Explores はインスタンス移行やバックアップ・リカバリ時に失われる可能性がある
  • Custom Calendars は Snowflake または BigQuery 接続のみ対応
  • Content Certification のステータスはコンテンツへの重要な編集時に自動的に取り消される

💡 ユースケース

📋 ユースケース 1: 非技術者による営業データの可視化

シナリオ: 営業チームのマネージャーが Google Sheets で管理している顧客リストと売上データを Looker で分析したい。LookML やデータベースの知識はない。

  1. Self-service Explores で Google Sheets URL を指定してデータをアップロード
  2. Looker が自動的にデータ型を検出し、Explore を生成
  3. Visualization Assistant を使って「売上を月別の棒グラフで表示して、上位 5 社を色分けして」と自然言語で指示
  4. 管理者が Content Certification で信頼性を認証し、チーム全体で共有

効果: BI 開発者への依頼を待つことなく、営業チーム自身が即座にデータ分析を開始でき、認証済みコンテンツとして組織的に活用可能

🗺️ ユースケース 2: 大規模データの地理的可視化

シナリオ: 物流チームが全国の配送拠点 30,000 件の所在地データを地図上に表示し、配送効率を分析したい。

  1. 管理者が Increased Row Limit を有効化し、地図チャートの行数制限を 50,000 行に設定
  2. Explore で配送拠点データをクエリし、地図チャートで可視化
  3. Conversational Analytics の Thinking モードで「配送効率が低い地域のパターンを分析して」と質問

効果: 従来の 5,000 行制限では表示できなかった全拠点データを一度に可視化し、AI 分析と組み合わせた高度なインサイト取得が可能

💰 料金・リージョン情報

💳料金

Gemini in Looker の各機能の料金は、利用するプラットフォーム (Looker (Google Cloud core) / Looker (original)) と契約形態により異なる。Preview 機能は現在無料で提供されているが、Preview 期間終了後は追加料金が発生する可能性がある。

🌍利用可能リージョン

Looker (Google Cloud core) インスタンスのリージョン可用性に準拠する。Conversational Analytics のデータ保存はインスタンスのリージョン内に制限されるが、転送中のデータはグローバルサービスで処理される場合がある。
